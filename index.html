Seismic-unix

Funciones básicas del Seismic Unix para realizar un modelamiento sismico (forward modelling)

Download this project as a .zip file Download this project as a tar.gz file

Primero: Instalación.

Para una correcta instalacion debemos instalar un compilador y preferiblemente (no obligatoriamente) actualizar el sistema.

En caso de usar ubuntu debe cambiar yum por apt-get 

$ sudo yum install gcc
$ sudo yum update

si ya tienes el compilador instalado y el sistema actualizado, podemos seguir:

Descargar archivos de instalación

Estos se pueden descargar desde la página del CWP/MINES o usando la siguiente línea de comando:

wget ftp://ftp.cwp.mines.edu/pub/cwpcodes/cwp_su_all_43R1.tgz

Como observamos, el archivo tiene extensión *.tgz, la debemos cambiar por *.tar.gz, se puede hacer modificando el nombre con F2 y escribiendolo manualmente o usando la siguiente línea de comando:

$ mv cwp_su_all_43R1.tgz cwp_su_all_43R1.tar.gz

Compilar e instalar el Seismix Un*x

Ahora, solo resta compilar los archivos necesarios, para simplificar esto se ha creado un Script que realiza automaticamente evitando errores comúnes de compilación. Ademas un video explicativo muestra la forma correcta de usarlo.

Link para Script

Link para Video

Al finalizar la instalación se debe escribir la siguiente línea de comando:

source ~/.bashrc

Errores Comúnes

Instalaciones fallidas deben ser borradas: simplemente eliminar la carpeta de instalación. Verificar que se hayan realizado todos los pasos anteriores. Verificar que el nombre del archivo comprimido coincida con lo explicado anteriormente.

Instalaciones fallidas: deben borrar el $CWPROOT, para esto deben escribir :

$ sudo gedit ~/.bashrc

y borrar la linea : export CWPROOT=
